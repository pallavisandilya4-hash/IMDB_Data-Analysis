This project analyzes the IMDB movie dataset using Python and Pandas.
The objective is to explore, clean, and analyze movie data to extract meaningful insights.
*Tools & Technologies Used:-
Python
Pandas
NumPy
*Dataset
The dataset contains information about movies such as:
Title
IMDB ID
Release Year
Adult Content Flag
Poster Path
Wikipedia Link
*Key Tasks Performed
Data loading and inspection
Handling missing values
Data cleaning and preprocessing
Filtering and sorting data
GroupBy operations
Applying functions
Exploratory Data Analysis (EDA)
*Key Insights
Analyzed movie release trends
Identified patterns in movie data
Performed structured data manipulation using Pandas
*What I Learned
Strong understanding of Pandas DataFrame operations
Data cleaning techniques
Real-world dataset handling
Practical implementation of Python for data analysis
